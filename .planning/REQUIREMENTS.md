# Requirements

## v1 Requirements

### Feed Presentation (FEED)
- [ ] **FEED-01**: User can view a seamlessly scrolling social media feed containing food posts.
- [ ] **FEED-02**: Posts display high-fidelity imagery and influencer-style deceptive captions.

### Discovery & Interaction (DISC)
- [ ] **DISC-01**: User can toggle a "Real View" via a physical-feeling flip animation on any post.
- [ ] **DISC-02**: Real View reveals true nutritional data (calories, sugar, fat, additives).
- [ ] **DISC-03**: Real View displays a pre-computed AI analysis of the caption's persuasive tactics.
- [ ] **DISC-04**: Real View displays a visual Bias Score illustrating structural misleadingness.

### Behavioral Capture (BEHAV)
- [ ] **BEHAV-01**: User can enter their age demographic upon session start seamlessly.
- [ ] **BEHAV-02**: User can select action choices on a post (e.g., "Would eat regularly?", "Pass").

### Scoring & Feedback (SCORE)
- [ ] **SCORE-01**: System locally compares user choices against pre-loaded survey baseline data.
- [ ] **SCORE-02**: System calculates and displays an evolving Awareness Score in real-time.
- [ ] **SCORE-03**: System displays algorithmic healthier alternatives for interacted posts.
- [ ] **SCORE-04**: System renders an end-of-session summary reflecting susceptibility insights.

## Out of Scope
- Backend Accounts / Authentication: Focus is on zero-friction onboarding and transient web simulation interaction.
- Live LLM Integration: Live analysis exposes rate limits, latency, and cost complexities. Focus on pre-seeded data analysis for immediacy.
